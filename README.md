# prntat

Flutter desktop + web starter for a print counter workflow.

## Features

- Desktop home screen with QR code that refreshes every 3 minutes
- Printer settings stub for color and mono printers
- Web order page stub with file upload, pricing, and invoice preview

## Run

Windows desktop:

```bash
flutter run -d windows
```

Web (mobile browser):

```bash
flutter run -d chrome --web-hostname 0.0.0.0 --web-port 8080
```
